# SQL_EDA_PROJECT
**Project Overview**:
This project explores global layoff trends using SQL through a comprehensive Exploratory Data Analysis (EDA).
The goal is to uncover insights around when, where, and in which industries layoffs occurred, as well as identify patterns across companies, countries, and time periods.
This analysis demonstrates strong SQL querying, data cleaning, aggregation, and analytical thinking skills.
